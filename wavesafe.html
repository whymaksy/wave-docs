wavesafe — what is it?

wavesafe is our proprietary, custom-built data protection framework designed to keep your personal lifestyle and financial records completely safe from unauthorized access and third-party tracking.

core pillars of wavesafe

  1. minimal data collection: we minimize data tracking to the absolute technical limit. only your username is visible for system routing.
  2. ws1.0 cryptographic protocol: a custom internal pipeline that manages locally-encrypted keys and strict data handling.
  3. end-to-end secure lifecycle: complete isolation of data on both your physical device storage and remote databases.

the ws1.0 encryption protocol

the ws1.0 protocol defines exactly how your data moves securely throughout its entire lifecycle:

data lifecycle pipeline:
  1. generation: the user creates data (a goal, task, or transaction record) inside the application.
  2. local isolation: the data is instantly compiled and stored into encrypted directories locally on the device using argon2id and aes-gcm algorithms.
  3. transport security: all data packets are tunneled through strict tls/https transport layers to completely prevent any man-in-the-middle (mitm) attacks.
  4. zero-knowledge cloud storage: the payload arrives at the database server as encrypted binary text. it is saved in its encrypted state, meaning no one—including system administrators—can read it.
